      I.        Objective of this Assignment: build a predictive model to find the weight-lifting quality.

 

   II.        Background:

This study uses the dataset file that was created by Human Activity Recognition Project. In this project several athletes were asked to perform some weight lifting exercises in 5 different ways but only one was the correct way of performing the lifting. The outcome of this project are two datasets, a training and testing datasets are made available for further study at [1]. Each of these datasets contains several attributes based on which the quality of weight-lifting exercises is determined. The last attribute classe is the outcome attribute that specifies of quality weight-lifting exercise using any one of these five levels A,B,C,D,E. These classe attribute information are supplied in the training dataset but not in the testing dataset. This study aims to build a predict model to find the classe for each of the 20 observations provided in the testing dataset.

  IV.        Loading Required Datasets

testing <- read.csv("pml-testing.csv")

training <- read.csv("pml-training.csv")

 

    V.        Data Preprocessing

a.     Removing first column : this ID attribute is removed to avoid interfering with ML algorithms

training <- training [c(-1)]

 

b.    Convert empty entries into NA

training[training==""] <- NA

 

c.    Remove the columns with more NA

##Create a vector to keep track of columns with more NAs

NAcols <- rep(FALSE, ncol(training)) ## default it to no NAs

 

## iterate through all columns and flag those with lots of NAs to get rid ##of them in the next step

for (i in 1:ncol(training)) {

if( sum(is.na(training[,i])) > 100) {

NAcols[i] <- TRUE

}

}

 

## eliminate columns with more NAs

training2 <- training[,!NAcols]

 

d.    Irrelevant column removal

training3 <- training2[,-c(1:6)]

After data preprocessing the training dataset contains 53 attributes, down from 160. One of these attributes classe is the outcome we are trying to predict, so the cleaned dataset contains 52 predictor attributes. Do the same for testing dataset

clean1 <- colnames(training3)

clean2 <- colnames(training3[, -53]) # already with classe column removed

 

testing1 <- testing[clean2]

 

dim(testing1)

 

  VI.        Create Data set for Cross Validation

inTrain <- createDataPartition(training3$classe, p = 0.7, list=FALSE)

train_subset <- training3[inTrain,]

crossval <- training3[-inTrain,]

 

VII.        Training using Random Forest Algorithm

model_control <- trainControl(method="cv", number=5, allowParallel=T, verbose=TRUE)

random_forest_model<-train(classe~.,data=train_subset,method="rf",trControl=model_control, verbose = FALSE)

 

VIII.        Model Validation on Cross Validation Dataset

pred_sub <- predict(random_forest_model, newdata=crossval)

## Extract the confusion matrix to assess model validity

confMat <- confusionMatrix(pred_sub, crossval$classe)

confMat$table

 

  IX.        Accuracy Calculation

accuracy <- sum((pred_sub==crossval$classe))/dim(crossval)[1]

 

    X.        Use Model for Prediction

answer=predict(random_forest_model, testing1)
